# Playing Card Assets

These playing-card image assets were provided by the server owner and are used by MariBot for card-game embeds.

Required filenames:

- `AS.png`, `AH.png`, `AD.png`, `AC.png`
- `2S.png` through `10S.png`, and the same pattern for `H`, `D`, `C`
- `JS.png`, `JH.png`, `JD.png`, `JC.png`
- `QS.png`, `QH.png`, `QD.png`, `QC.png`
- `KS.png`, `KH.png`, `KD.png`, `KC.png`
- `Card-back.png`

Suit codes:

- `S`: spade
- `H`: heart
- `D`: diamond
- `C`: club

If this folder is missing in deployment, MariBot will generate simple fallback card images at runtime so card games do not crash.
